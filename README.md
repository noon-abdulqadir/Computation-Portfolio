# Computational Work Porfolio
This repository is sample computational work. It containts the following directories:
1. ``forked_repositories``: Select existing projects, forked and edited. These includes:

     1. ``plot_metric_fork``, which was edited to work with newer versions of pandas
     2. ``specification_curve_fork``, which was edited to allow annotation and specific coloring.
2. ``helper_classes``: Select .py helped scripts. These are:
     1. ``multilevel_modeling``, which performs descriptives, visualization, multilevel modeling, and model comparisons
     2. ``necessary_data_classes``, which obtains data from official sources and preprocesses them
3. ``qualtrics_js``: Javascript, HTML, and CSS based project that runs a qualtrics embedded experiment. The scripts randomize across 12 conditions with 3 trials each
